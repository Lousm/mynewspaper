#### 简介
- 基于Newspaper做了符合业务逻辑的改进
	- ``` text ``` 升级 ``` all_text ```
	- ``` article_html ``` 升级 ``` all_ article_html ```
- 测试Case
	- 测试 ``` text ``` 内容缺失
	      > demo file
	      
		> examples/text_demo.py
	- 测试 ``` article_html ```  内容缺失
		> demo file
	      
		> examples/article_html_demo.py
 - 安装使用
 > pip install git+https://github.com/Lousm/mynewspaper.git
